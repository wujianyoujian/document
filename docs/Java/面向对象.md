# 面向对象
* 三大特征
> **封装**, **继承**, **多态**

```java
package com.wuzu.类;

class Human {
    // 属性(成员属性)
    String name;
    int age;
    boolean ifMan;
    // 方法
    public void walk() {
        System.out.print("Walk");
    }
}

public class Person {
    public static void main(String[] args) {
        // 局部变量, 声明在方法内, 方法形参, 代码块, 构造器形参, 构造器内部变量内
        Human h1 = new Human();
        h1.walk();
    }
}
```

---

注意:
  在调用局部变量之前需要显示赋值
